Student SMS

The SMS module allows you to broadcast SMS (short messages) to parents.

Student

  1. Select the search criteria.
  2. Click the Search button. Students belonging to the selected category are populated.
  3. Select the student/s. Selected students are highlighted. 
    1. Click Ctrl+A to select all the students. 
    2. Click on students while pressing Ctrl to select multiple students. 
  4. Click the right arrow button. The selected students are shown on the Selected student(s) box.
  5. Enter the message in the Message textbox.  
  6. Click the Send button to send the message. 
Note:
  • SMS notifications can be sent to the parents of the students selected from following categories.
    • Admission no
    • Grade
    • Class
    •  Sport
    • Club and society
    • Prefect Type
    • Status (current, temporary leave, suspended)
    • Subject
    • Last name
  • Message should not exceed 160 characters.
  • Students whose parents have exceeded their SMS quota are shown in a grey background.
  • When parents exceed their allocation of SMS quota, notifications are sent using the SMS quota allocated to the school.
  • Search results are not generated when both the user and the school have exceeded the SMS quota.
