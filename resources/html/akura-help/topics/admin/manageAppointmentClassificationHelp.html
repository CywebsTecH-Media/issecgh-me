Manage Appointment Classification

Navigation: Select Reference Module->Manage Appointment Classification

Using this page you can handle Appointment Classification related information in the system. 

View Appointment Classification

This is the homepage for the Appointment Classification module. This page displays a list of Appointment Classification currently available in the system.

Add New Appointment Classification 

  1. Click the Add New Appointment Classification button.
  2. Enter the new Appointment Classification name.
  3. Click the Save button to save the Appointment Classification or click the Cancel button to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Appointment Classification 

  1. Click the Edit Appointment Classification button. The selected row is highlighted in yellow. 
  2. Edit the Appointment Classification name.
  3. Click the Save button to save the Appointment Classification or click the Cancel button to cancel the changes. 

Delete Appointment Classification 

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. Click OK to delete the selected record or click the Cancel button to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
