Enter Verification Code

Navigation: Login->Enter Verification Code

Using this page, you can strengthen the security of parent user accounts by entering the verification code that was sent to your mobile phone by the ABM SMS system.

  1. Entet the Verification Code.
  2. Click the Verify button.
Note:
  • The parent mobile phone number must be available in the Parent Details page.
