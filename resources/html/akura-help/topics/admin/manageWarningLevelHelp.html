Manage Warning Levels

Navigation: Select Reference Module->Manage Warning Levels

Using this page, you can view, add, edit and delete warning level related information in the system.

View Warning Levels Details

This is the homepage for the general warning levels in the school. It displays a list of warning levels available in the system.

Add New Warning Levels

  1. Click the Add New Warning Level button.
  2. Enter the Warning Level.
  3. Select a Colour.
  4. Click the Save button to add the warning level or click Cancel to cancel the add operation.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Warning Levels

  1. Click the Edit Warning Level button. The selected row is highlighted in yellow. 
  2. Edit the Warning Level.
  3. Change the Colour.
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Warning Levels

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
