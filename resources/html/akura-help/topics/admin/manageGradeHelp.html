Manage Grades

Navigation: Select Reference Module->Manage Grades

Using this page, you can view, add, edit and delete grade related information in the system.

View Grade Details

This is the homepage for the Grade section. It displays a list of grades available in the system.

Add New Grade

  1. Click the Add New Grade button.
  2. Enter the Grade Name.
  3. Select the classes from the check boxes.
  4. Click the Save button to commit the add operation. 
  5. Click the Cancel button to go to the Grade Details page.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Grade

  1. Click the Edit Grade button of the relevant grade. The selected grade is highlighted in yellow. 
  2. Edit the Grade Name of the Classes.
  3. Click the Save button to save the changes.
  4. Click the Cancel button to go to the Grade Details page.

Delete Grade

  1. Click the Delete button of the relevant grade. The selected grade is highlighted in yellow. 
  2. A confirmation message is displayed. Click OK to delete the grade or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
