Manage Donation Type

Navigation: Select Reference Module->Manage Donation Type

Using this page, you can view, add, edit and delete donation types in the system.

View Donation Type Details

This is the homepage for the Donation Type section. It displays a list of donation types available in the system.

Add New Donation Type

  1. Click the Add New Donation Type button.
  2. Enter the Donation Type.
  3. Click the Save button to add the donation type.
  4. Click the Cancel button to go to the Donation Type Details page.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Donation Type

  1. Click the Edit Donation Type button. The selected row is highlighted in Yellow.
  2. Edit the Donation Type.
  3. Click the Save button to save the changes.
  4. Click the Cancel button to go to the Donation Type details page.

Delete Donation Type

  1. Click the Delete button. A confirmation message is displayed. The selected row is highlighted in yellow.
  2. Click OK to confirm the deletion. Click Cancel to go to the Donation Type Details page. 
Note: Data shared by other modules cannot be deleted.
