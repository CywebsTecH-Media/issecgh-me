Manage Educational Qualifications

Navigation: Select Reference Module->Manage Educational Qualifications  

Using this page you can view, add, edit and delete educational qualification related information in the system. The storage of educational qualification related information is important to identify the educational qualifications of the staff members.

View Educational Qualification Details

This is the homepage for the Educational Qualification section. It displays a list of Educational Qualifications available in the system.

Add New Educational Qualification 

  1. Click the Add New Educational Qualification button. The selected row is highlighted in yellow. 
  2. Select the academic title from the drop-down list.
  3. Select the qualification type from the radio buttons.
  4. Enter the Study Area.
  5. Click the Save button to add the educational qualification or click Cancel to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Educational Qualification 

  1. Click the Edit Educational Qualification button. The selected row is highlighted in yellow. 
  2. Select the academic title from the drop-down list.
  3. Select the qualification type from the radio buttons.
  4. Edit the Study Area.
  5. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Educational Qualification 

  1. Click the Delete button.
  2. Click OK to delete the selected record or click Cancel to go to the Educational Qualification Details page. 
Note: Data shared by other modules cannot be deleted.
