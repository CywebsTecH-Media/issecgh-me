Add/Update User Role Details

Navigation: Select User Management Module->Manage System Users->Add New System User->System User Details
Using this page, you can add or update user role details.

Add User Role Details

  1. Click the Add New Role button.
  2. Provide the following required details:
    • Enter the role name.
    • Assign or revoke privileges from the check boxes.
  3. Click the Save button to save the details. 
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Click the Reset button to reset the fields.
  • Children privileges and User Management Module privileges cannot be assigned to the customized user roles. 

Edit User Role Details

  1. Click the Edit button of the relevant user role.
  2. Update the user roles details in the User Role Details page.
  3. Click the Save button to save the details or click Cancel to cancel the changes.
