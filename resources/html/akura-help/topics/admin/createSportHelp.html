Manage Sports

Navigation: Select Reference Module->Manage Sports

Using this page, you can view, add, edit and delete sports in the system.

View Sports Details

This is the homepage for the Sports section. It displays a list of sports available in the system.

Add New Sport

  1. Click the Add New Sport button.
  2. Enter the name of the sport.
  3. Click the Save button to save the details or click Cancel to cancel the add operation. 

Note: Fields marked with an asterisk (*) are mandatory. 

Edit Sport

  1. Click the Edit Sport button of the relevant sport. The selected row is highlighted in yellow. 
  2. Edit the Sport field.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Sport

  1. Click the Delete button of the sport.
  2. A confirmation message is displayed. Click OK to delete the selected sport or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
