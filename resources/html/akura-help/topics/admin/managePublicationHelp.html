Manage News and Events

Navigation: Select Reference Module->Manage News and Events

Using this page, you can view, add, edit and delete news and events in the system.

View New News And Events

This is the homepage for the News & Events section. It displays a list of news and events recorded in the system.

Add New News And Events

Note: Adding news and event imformation will send an SMS notification to the parents who have subscibed to the SMS service.
  1. Click the Add New News And Events button.
  2. Provide the following details.
    • Header
    • Message
    • Upload Image
  3. Provide an expire date using the Calendar button.
  4. Select the Type.
  5. Check the "Notify via SMS" to send SMS notifications to parents.
  6. Click the Save button to save the details or click Cancel to cancel the add operation.
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Only the parents who have subscribed to SMS notifications will receive messages. 
  • SMS notifications are sent only from the allocated individual (parent) SMS quota.

Edit News and Events

  1. Click the Edit News and Item button of a News and Events item from the list. The selected row is highlighted in yellow.
  2. Edit the data.
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete News and Events

  1. Click the Delete button of a News and Events item. The selected row is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion.
