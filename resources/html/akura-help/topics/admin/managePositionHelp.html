Manage Membership Positions

Navigation: Select Reference Module->Manage Membership Positions

Using this page, you can view, add, edit and delete membership positions related information in the system.

View Membership position Details

This is the homepage for the Membership Positions section. It displays a list of membership positions available in the system.

Add New Membership Positions 

  1. Click the Add New Membership Position button.
  2. Enter the Membership Position in the given field.
  3. Click the Save button to save the details or click Cancel to cancel the operation.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Membership Positions

  1. Click the Edit Membership Position button of the membership position. The selected row is highlighted in yellow.
  2. Edit the Membership Positions field.
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Membership Positions

  1. Click the Delete button of the relevant membership position. The selected row is highlighted in yellow.
  2. A confirmation message will be displayed.
  3. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
