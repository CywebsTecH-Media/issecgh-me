Manage Exams

Navigation: Select Reference Module->Manage Exams

Using this page, you can view, add, edit and delete exam related data in the system.

View Exams Details

This is the homepage for the Exams section. It displays a list of exams available in the system.

Add New Exam 

  1. Click the Add New Exam button.
  2. Select the Grade relevant for the exam from the drop-down list. 
  3. Enter the name of the exam in the given field..
  4. Select the Mark Type as grade or marks. 
  5. Click the Save button to save the changes or click Cancel to cancel the changes. 

Note:

  • Fields marked with an asterisk (*) are mandatory.
  • When adding an exam it should be separated for different streams, eg. for A/L , A/L Science, A/L Commerce, A/L Arts.
  • Then from Exam Subjects, the UI allocates subjects for those exams separately.

Edit Exam

  1. Click the Edit Exam button of a relevant exam. The selected row is highlighted in yellow.
  2. Edit the data.
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Exam

  1. Click the Delete button of a relevant exam. The selected row is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
