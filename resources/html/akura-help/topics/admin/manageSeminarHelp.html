Manage Seminars

Navigation: Select Reference Module->Manage Seminars

Using this page, you can view, add, edit and delete seminar related information in the system. 

View Seminar

A list of seminars available in the system is displayed.

Add New Seminar

  1. Click the Add New Seminar button.
  2. Enter the name of the seminar.
  3. Click the Save button to add the seminar. 
  4. Click Cancel to cancel the add operation.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Seminar

  1. Click the Edit Seminar button. The selected row is highlighted in yellow. 
  2. Edit the name of the seminar. 
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Seminar

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. Click the OK button to confirm the deletion or click Cancel to cancel the deletion.  
Note: Data shared by other modules cannot be deleted.
