View Audit Details

Navigation: Select Reference Module-> View Audit Details 
Using this page, you can view audit details. With this page, administrators can view the additions and changes made by various users to the system. 

View Audit Details

  1. Provide the following details:
    • Audit Category
    • User
    • From Date
    • To Date
    • Grade
    • Class
    • Year
    • Term/Exam/Assignment
  2. Click the Search button to view search results.
Note: Fields marked with an asterisk (*) are mandatory.
