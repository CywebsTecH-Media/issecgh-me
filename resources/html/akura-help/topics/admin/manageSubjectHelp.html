Manage Subjects

Navigation: Select Reference Module->Manage Subjects

Using this page, you can search and then add, edit and delete subjects in the system.

Search Subject

  1. Enter the full name or partial name of a subject.
  2. Click the Search button to get search results. 

View Subject Details

This is the homepage for the Subject section which displays a list of subjects available in the system.

Add New Subject

  1. Click the Add New Subjects button.
  2. Enter the Subject Name.
  3. Provide a Short Name. 
  4. Select a Stream.
  5. Provide the Subject Code given by the government.
  6. Click the Save button to add the subject or click Cancel to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Subject

  1. Click the Edit Subject button. The selected record is highlighted in yellow.
  2. Edit the Subject Name, Short Name, Code and the Stream.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Subject

  1. Click the Delete button. The selected record is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
