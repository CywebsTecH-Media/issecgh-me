Manage Staff Leave Types

Navigation: Select Reference Module->Manage Staff Leave Types

Using this page, you can manage staff leave in the system.

View Staff Leave Types Details

This is the homepage for the general staff leave types in the school. It displays a list of staff leave types available in the system.

Add New Staff Leave Type

  1. Click the Add New Staff Leave Type button. 
  2. Enter the Staff Leave Type and the maximum number of staff leave.
  3. Select the specific gender to the staff leave type from the radio buttons.
  4. Click the Save button to save the details or click Cancel to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Staff Leave Type

  1. Click the Edit Staff Leave Type button. The selected row is highlighted in yellow. 
  2. Edit the Staff Leave Type and the maximum number of staff leave.
  3. Change the gender specific to the staff leave type from the radio buttons.
  4. Click the Save button to save the changes or click Cancel to cancel the changes. 
Note: Leave types that are in use cannot be edited.

Delete Staff Leave Type

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Leave types that are in use cannot be deleted.
