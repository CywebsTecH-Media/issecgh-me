Manage Classes

Navigation: Select Reference Module->Manage Classes

Using this page, you can view, add, edit and delete classes in the system.

View Class Details

This is the homepage for the Class section. It displays a list of classes available in the system.

Add New Class

  1. Click the Add New Class button.
  2. Enter the Class Name.
  3. Click the Save button to save the details or click Cancel to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Class

  1. Select a Class from the list to edit. The selected class is highlighted in yellow. 
  2. Click the Edit Class button of the relevant Class.
  3. Edit the data.
  4. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Class 

  1. Select a Class from the list to delete. The selected class is highlighted in yellow. 
  2. Click the Delete button of the relevant Class. A confirmation message is displayed.
  3. Click OK to confirm the deletion or click Cancel to cancel the deletion. 

Note: Data shared by other modules cannot be deleted.
