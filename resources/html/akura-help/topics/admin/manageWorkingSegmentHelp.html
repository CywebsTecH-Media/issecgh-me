Manage Working Segment

Navigation: Select Reference Module->Manage Working Segment

Using this page, you can view, add, edit and delete working segments in the system.

View Working Segments

This is the homepage for the working segment section. It displays a list of working segments available in the system.

Add New Working Segment

  1. Click the Add New Working Segment button.
  2. Enter a Working Segment.
  3. Click the Save button to save the details or click Cancel to cancel the changes.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Working Segment

  1. Click the Edit Working Segment button of a particular working segment. The selected row is highlighted in yellow.
  2. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Working Segment

  1. Click the Delete button of a particular working segment. The selected row is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
