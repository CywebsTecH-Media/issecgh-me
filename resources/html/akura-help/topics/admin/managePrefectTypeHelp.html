Manage Prefect Types

Navigation: Select Reference Module->Manage Prefect Types

Using this page, you can view, add, edit and delete the prefect types in the system.

View Prefect Types

This is the homepage of the Prefect Types section. It displays a list of prefect types available in the system.

Add New Prefect Type Procedure

  1. Click the Add New Prefect Type button.
  2. Enter the name of a prefect type.
  3. Click the Save button to save the details or click Cancel to cancel the add operation.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Prefect Type Procedure

  1. Click the Edit Prefect Type button of a relevant prefect type. The selected row is highlighted in yellow.
  2. Edit the name of the prefect type.
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Prefect Type Procedure

  1. Click the Delete button of a relevant prefect type. The selected row is highlighted in yellow. 
  2. A confirmation message is displayed. 
  3. Click the OK button to confirm the deletion or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
