Manage Sports Categories

Navigation: Select Reference Module->Manage Sports Categories

Sports categories are created by combining a sport and an age group. Using this page, you can view the sports categories and then add, edit and delete sports categories.

View Sport Category Details

This is the homepage for the Sports Categories section. It displays a list of sports categories available in the system.

Add New Sport Category

  1. Click the Add New Sport Category button.
  2. Select the following field:
    • Sport
    • Sport Age Group
  3. Click the Save button to add the new sport category or click Cancel to cancel the process.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Sport Category

  1. Click the Edit Sport Category button of a selected sports category.
  2. Edit the Sport field and the Sports Age Group field.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Sport Category

  1. Click the Delete button of a selected sports category.
  2. A confirmation message is displayed. Click OK to delete the selected sports category or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
