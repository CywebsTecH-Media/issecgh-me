Manage Appointment Nature

Navigation: Select Reference Module->Manage Appointment Nature

Using this page, you can view, add, edit and delete appointment nature related information in the system. The storage of appointment nature related information is important to identify the appointment nature (e.g., Temporary, Permanent, Part-time, Volunteer) of the staff members.

View Appointment Nature Details

This is the homepage for the Appointment Nature section. It displays a list of appointment natures available in the system.

Add New Appointment Nature 

  1. Click the Add New Appointment Nature button. 
  2. Enter the Appointment Nature.
  3. Click the Save button to add the appointment nature or click the Cancel button to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Appointment Nature 

  1. Click the Edit Appointment Nature button. The selected row is highlighted in yellow. 
  2. Edit the Appointment Nature.
  3. Click the Save button to edit the appointment nature or click the Cancel button to cancel the changes.

Delete Appointment Nature 

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. Click OK to delete the selected record or click Cancel to go to the Appointment Nature Details page. 
Note: Data shared by other modules cannot be deleted.
