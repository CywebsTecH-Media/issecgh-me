Manage Special Events

Navigation: Select Reference Module->Manage Special Events

Using this page, you can view, add, edit and delete special events in the system. The page displays a list of special events available in the system.

View Special Events

This is the homepage for the Special Event section. It displays a list of special events available in the system.

Add New Special Events and Participants List

  1. Click the Add New Special Events and Participants button.
  2. Select the following fields:
    • Name of the Special Event
    • Description of the Special Event
    • Date of the Special Event
    • Participant Category of the Special Event
  3. Select participants from the Participants list. 
  4. Click the Save button to add the event or click Cancel to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Special Event and Participants List

  1. Click the Edit Special Event and Participants button of a selected special event. The selected row is highlighted in yellow.
  2. Edit the fields.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Special Event

  1. Click the Delete button of a selected special event. The selected row is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
