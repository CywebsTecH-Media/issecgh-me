Manage Assignments

Navigation: Select Reference Module->Manage Assignment

Using this page, you can view, add, edit and delete assignments. Teachers can assign 

View Assignment Details

This is the homepage for the Assignment section. It displays a list of assignments available in the system.

Add New Assignment

  1. Click the Add New Assignment button.
  2. Provide the following details:
    • Grade
    • Subject
    • Submission Date using the Calendar icon 
    • Assignment Name
    • Description of the assignment
    • Assignment Marks Type
  3. Click the Save button to add the new assignment or click Cancel to cancel the process.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Assignment

  1. Click the Edit Assignment button of an assignment. The selected row is highlighted in yellow.
  2. Edit the data set.
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Assignment

  1. Click the Delete button of an assignment. The selected row is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to delete the selected assignment or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
