Manage Civil Status

Navigation: Select Reference Module->Manage Civil Status

Using this page, you can view, add, edit and delete civil status related information in the system. 

View Civil Status Details

This is the homepage for the Civil Status section. It displays a list of civil statuses available in the system.

Add New Civil Status

  1. Click the Add New Civil Status button.
  2. Enter the Civil Status.
  3. Click the Save button to add the Civil Status or click the Cancel button to go to the Civil Status Details page.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Civil Status

  1. Click the Edit Civil Status button. The selected row is highlighted in yellow. 
  2. Edit the Civil Status.
  3. Click the Save button to save the changes or click the Cancel button to cancel the changes.

Delete Civil Status

  1. Click the Delete button. A confirmation message is displayed. 
  2. Click the OK button to confirm the deletion or click Cancel to go to the Civil Status Details page.
Note: Data shared by other modules cannot be deleted.
