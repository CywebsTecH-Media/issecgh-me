Manage Professional Qualifications

Navigation: Select Reference Module->Manage Professional Qualification

Using this page, you can view, add, edit and delete professional qualifications in the system. The storage of professional qualification related information is important to identify the professional qualification of the staff member.

View Professional Qualification Details

This is the homepage for the professional qualification section. It displays a list of professional qualifications available in the system.

Add New Professional Qualification

  1. Click the Add New Professional Qualification button.
  2. Enter the Professional Qualification.
  3. Click the Save button to add the professional qualification or click Cancel to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Professional Qualification

  1. Click the Edit Professional Qualification button. The selected row is highlighted in yellow. 
  2. Edit the Professional Qualification.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Professional Qualification

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
