Manage Sports Age Groups

Navigation: Select Reference Module->Manage Sports Age Groups

Using this page, you can view, add, edit and delete sports age groups in the system.

View Sports Age Details

This is the homepage for the Sports Age section. It displays a list of sports age categories available in the system.

Add New Sports Age Group

  1. Click the Add New Sports Age Group button.
  2. Enter the Sports Age Group in the given field.
  3. Click the Save button to save the details or click Cancel to cancel the add operation.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Sports Age Group

  1. Click the Edit Sports Age Group button of the sports age group.The selected row is highlighted in yellow.
  2. Edit the Sports Age Group field.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Sports Age Group

  1. Click the Delete button of the sports age group. The selected row is highlighted in yellow. 
  2. A confirmation message is displayed. Click OK to delete or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
