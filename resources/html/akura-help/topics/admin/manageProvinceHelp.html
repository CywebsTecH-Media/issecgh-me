Manage Provinces

Navigation: Select Reference Module->Manage Provinces

Using this page, you can search for a province and then add, edit and delete provinces in the system.

Search Province

  1. Enter the full name or partial name of a province.
  2. Click the Search button to view search results. 

View Province Details

This is the homepage for the Province section. It displays a list of provinces available in the system.

Add New Province

  1. Click the Add New Province button.
  2. Enter the name of the province. 
  3. Click the Save button to add the province.
  4. Click the Cancel button to go to the Province Details page.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Province

  1. Click the Edit Province button. The selected row is highlighted in yellow. 
  2. Edit the name of the province. 
  3. Click the Save button to save the changes. 
  4. Click the Cancel button to go to the Province Details page.

Delete Province

  1. Click the Delete button. A confirmation message is displayed. 
  2. Click OK to confirm the deletion or click Cancel to cancel the changes. 
Note: Data shared by other modules cannot be deleted.
