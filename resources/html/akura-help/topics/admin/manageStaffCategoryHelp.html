Manage Staff Category Information

Navigation: Select Reference Module->Manage Staff Service Category

Using this page, you can view staff category details and add, edit and delete staff categories in the system.

Staff Category Details

This is the homepage for the Staff Category section. It displays a list of staff categories available in the system.

Add New Staff Category 

  1. Click the Add New Staff Category button.
  2. Enter the Staff Category.
  3. Select a Staff Type from radio buttons.
  4. Click the Save button to add the staff category or click the Cancel button to go to the Staff Category Details page.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Staff Category 

  1. Click the Edit Staff Category button of a category. The selected row is highlighted in yellow. 
  2. Edit the Staff Category and the Staff Type.
  3. Click the Save button to save the changes or click the Cancel button to go to the Staff Category Details page.

Delete Staff Category 

  1. Click the Delete button of a staff category. The selected row is highlighted in yellow. 
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to go to the Staff Category Details page.
Note: Data shared by other modules cannot be deleted.
