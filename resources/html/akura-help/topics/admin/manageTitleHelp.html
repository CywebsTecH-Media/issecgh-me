Manage Academic Titles

Navigation: Select Reference Module->Manage Academic Titles

Using this page, you can view, add, edit and delete title related information in the system. 

View Title Details

This is the homepage for the Academic Title section. It displays a list of titles available in the system.

Add New Title

  1. Click the Add New Title button.
  2. Enter the Title.
  3. Click the Save button to add the Title.
  4. Click the Cancel button to cancel the add operation.
Note: Fields marked with an asterisk (*) are mandatory.

Edit Title

  1. Click the Edit Title button. The selected row is highlighted in yellow. 
  2. Edit the Title.
  3. Click the Save button to save the changes or click the Cancel button to cancel the changes.

Delete Title

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. Click the OK button to confirm the deletion or click the Cancel button to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
