Manage Clubs & Societies

Navigation: Select Reference Module->Manage Clubs & Societies

Using this page, you can view, add, edit and delete clubs and societies.

View Clubs & Societies

This is the homepage for the Clubs & Societies section. It displays a list of clubs and societies available in the system.

Add New Club or Society

  1. Click the Add New Club or Society button.
  2. Fill the following fields:
    • Club or Society Name
    • Description
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Club or Society

  1. Click the Edit Club or Society button. The selected row is highlighted in yellow. 
  2. Edit the Club or Society data.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Club or Society

  1. Click the Delete button. The selected row is highlighted in yellow. 
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
