Manage Holidays

Navigation: Select Reference Module->Manage Holidays

Using this page, you can view, add, edit and delete holidays for a school year.

View Holidays

This is the homepage for the Holidays section. It displays a list of holidays available in the system. Select the year to view the list of holidays for that particular year.

Add New Holiday

  1. Click the Add New Holiday button.
  2. The Add New Holiday form is displayed at the bottom of the current page.
  3. Fill the following fields:
    • Start Date
    • End Date
    • Reason
  4. Click the Save button to save the holiday or click Cancel to cancel the addition. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Holiday

  1. Click the Edit Holiday button of a relevant holiday. The selected row is highlighted in yellow.
  2. The Edit Holiday form is displayed. Edit the Start Date, End Date and Reason and click the Save button to save the changes or click Cancel to cancel the changes.

Delete Holiday

  1. Click the Delete button of a Holiday. The selected row is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to delete the holiday or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
