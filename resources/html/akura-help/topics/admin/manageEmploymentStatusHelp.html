Manage Employment Status

Navigation: Select Reference Module->Manage Employment Status 

Using this page, you can view, add, edit and delete employment status related information in the system.

View Employment Status

This is the homepage for the Employment Status section. It displays a list of employment statuses available in the system.

Add New Employment Status

  1. Click the Add New Employment Status button.
  2. Enter the Employment Status.
  3. Click the Save button to add the employment status or click Cancel to cancel the add operation. 
Note: Fields marked with an asterisk (*) are mandatory.

Edit Employment Status

  1. Click the Edit Employment Status button of a particular employment status. The selected row is highlighted in yellow.
  2. Edit the data.
  3. Click Save to save the changes or click Cancel  to cancel the changes.

Delete Employment Status

  1. Click the Delete button of a particular employment status. The selected row is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to delete the employment status or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
