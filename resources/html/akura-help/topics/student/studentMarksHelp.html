Student Marks

Navigation: Select Student Module->Student Mark Entry

Using this page, you can maintain student marks. Marks can be Term Marks or Sub-term Marks (monthly marks). Maintaining student marks is a two-step process:

  • Search for a class.
  • Enter Term Marks or Sub-term Marks (monthly marks).

Note: You must assign subjects to students before assigning marks.

Search for Class

  1. Select the year from the Year drop-down list.
  2. Select the class from the Select Class drop-down list.
  3. Select the term from the Select Term drop-down list.
  4. Select Monthly Grades or Term Grades from the Select Grade drop-down list.
  5. Click the Search Class button. The students and subjects for the selected class are displayed.

Note:

  • Fields marked with an asterisk (*) are mandatory.
  • Suspended student records are highlighted. 

Enter Term Marks

  1. For Term Grades, enter the marks as a percentage.
  2. Select the Click the check box to confirm completion of marks entry for all subjects check box.
  3. Click Reset to discard all unsaved changes.
  4. Click Save to save the marks or click Cancel to cancel the operation.
Note:
  • The Click the check box to confirm completion of marks entry for all subjects check box should be selected only after entering all the term marks.
  • Users cannot enter term marks for students on temporary leave.
  • Users have the capability to enter marks for suspended students.
  • Term marks can be entered by the admin user/class teacher or any teacher of the school.
  • Marking can be completed only by an admin user or a class teacher.
  • Only the admin user can edit the marks once marking is completed.

Import/Export Term Marks

       File Upload Guidelines
  • The user must always use the Term Mark_[term]_[class]_[year].xls template which can be downloaded by clicking the Export button.
  • The capacity of the XLS file should not exceed 2MB.
  • Format of the XLS file cannot be changed.
  • Admission Number and Student Name fields are mandatory. Admission numbers should match with existing data in the database.
  • Following rules should be applied when entering marks to the excel sheet:
    • Marks should be greater than or equal to 0.00
    • Fractional values should be rounded to two decimal places.
    • Any variation of ‘Ab’ can be entered.

  1. Click Export to download an excel sheet with existing data. If there are no existing data, only the template with field names will be downloaded.
  2. Click Import to upload marks into the system through the excel sheet. A new pane will be displayed to upload the document.
  3. Browse the XLS file.
  4. Click the Upload button to upload data to the system.

Enter Sub-term Marks (Monthly Marks)

  1. For Monthly Grades, enter the marks as a percentage. 
  2. Click Reset to discard all unsaved changes.
  3. Click Save to save the marks or click Cancel to cancel the operation.

Note:

  • The admin user or a teacher can enter monthly marks.
  • Only the admin user can edit monthly marks once marking is completed.
