Activate/Extend Temporary Leave Student Service

Navigation: Select Student Module->Search Student->Activate Temporary Leave Student

Using this page, you can revoke the temporary leave status of a student or extend the temporary leave. 

Activate Temporary Leave Student

  1. Select the student status as Temporary Leave from the drop-down list.
  2. Click the Search button.
  3. Click the Activate Temporary Leave Student  button. The Activate/Extend Temporary Leave Student Service window is opened.
  4. Click the Activate button. 
  5. Select the date of activation using the Calendar button.
  6. Select the Class and Year from the drop-down lists.
  7. Click the Activate button to confirm the activation.

Extend Temporary Leave

  1. Select the student status as Temporary Leave from the drop-down list.
  2. Click the Search button.
  3. Click the Activate Temporary Leave Student  button. The Activate/Extend Temporary Leave Student Service window is opened.
  4. Click the Extend button. 
  5. Select the Date To which the temporary should be extended, using the Calendar button (Reason and Date From fields are disabled).
  6. Click the Save button to confirm the extention.
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Parent/guardian's e-mail addresses should be pre-defined to automatically inform the parents.
