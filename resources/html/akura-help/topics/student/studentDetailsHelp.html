Student Details

Navigation: Select Student Module->Search Student->Edit->Student Details.

Using this page, you can add student details to the system.

  1. Complete the General Information section. 
  2. Enter the previous school details, health issues information and emergency contact information if required (not mandatory).
  3. Click Save to save the details or click Cancel to cancel the add operation.

Note:

  • Fields marked with an asterisk (*) are mandatory.
  • Before adding a student, make sure that the required reference data (Religion, Study Medium, Nationality, School House, Method of Travel and City information) has been entered to the system through the Reference module.
  • Clicking the Reset button will reset the Student Details page.
  • The Attendance progress bar is used to display a summary of the student's attendance details for the selected year.
  • Clicking the Home link will take you to the Homepage.
  • Clicking the Student Search link will take you to the Student Search page.
  • Clicking the Print Student Summary link will allow you to view and print the Student Summary Report.
  • The Print Student Report Card link will only be available for students whose term marks have been finalized.
