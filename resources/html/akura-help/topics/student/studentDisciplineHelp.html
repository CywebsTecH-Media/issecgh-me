Student Discipline

Navigation: Select Student Module->Search Student->Edit->Discipline

Using this page, you can manage student discipline.

Note: 

  • Fields marked with an asterisk (*) are required. 
  • Existing disciplinary records will be listed on the Student Discipline page.
  • If subscribed, an SMS notification will be sent to the parent/guardian.
  • Warning Levels should be defined through the Reference module in order to assign disciplinary actions. 
  • The Attendance progress bar is used to display a summary of the student's attendance details for the selected year.
  • The Religious Activities progress bar is used to display a summary of the student's religious activities for the selected year.
  • The Academic Life progress bar is used to display a summary of the student's academic life for the selected year.
  • Disciplinary action allocated for a student suspension process cannot be deleted.
  • Click the Home link to go to the Homepage. 
  • Click the Student Search link to go to the Student Search page.

Add Student Discipline details

  1. To add student discipline details, click the Add button.
  2. Enter a Comment.
  3. Select the Warning Level.
  4. Select the Date.
  5. Select Yes or No for Informed Parents.
  6. Click Save to save the details or click Cancel to cancel the add operation.

Edit Student Discipline Details

  1. To edit student discipline details, click the Edit button in the row section.
  2. Make changes as required.
  3. Click Save to save the changes or click Cancel to cancel the changes.

Delete Student Discipline Details

  1. To delete student discipline details, click the Delete button in the row section.
  2. A confirmation message is displayed. Click OK to delete the student discipline details or click Cancel to cancel the deletion.
