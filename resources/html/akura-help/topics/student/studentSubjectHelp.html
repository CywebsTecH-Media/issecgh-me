Assign Subjects to Students

Navigation: Select Student Module->Student Subjects.

Using this page, you can assign subjects to students. Assigning subjects to students is a two-step process:

  • Search for a class.
  • Assign subjects to the students in the class.

Search for Class

  1. Select the class using the Select Class drop-down list.
  2. Select the year using the Select Year drop-down list.
  3. Select the term using the Select Term drop-down list.
  4. Click the Search Class button. The students and subjects for the selected class will be displayed along with any current subject assignments.

Note:

  • Fields marked with an asterisk (*) are mandatory.
  • Suspended students or students on temporary leave are not listed under classes.

Assign Subjects to Students

  1. To assign subjects, do one of the following:
    • To assign subject(s) to a student, select the relevant subject check box(es) next to the student's name.
    • To remove subject assignments, clear the relevant subject check box(es) next to the student's name.
  2. Click Reset to discard all unsaved changes.
  3. Click Save to assign the subjects, or click Cancel to cancel the operation.

Note:

  • Only the admin and the class teachers are allowed to assign subjects to students.
  • The user must assign students to a class before assigning subjects to them.
  • The user cannot assign subjects to students for terms where the term marks have already been completed.
  • Core subjects are automatically assigned to the students.
