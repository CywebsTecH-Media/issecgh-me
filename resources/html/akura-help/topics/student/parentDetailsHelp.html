Parent Details

Navigation: Select Student Module->Search Student->Edit->Parent Details

Using this page, you can maintain parent/guardian details of students.

Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Existing parent details will be listed on the Parent Details tab. 
  • Before adding the religion, employment status, working segment, donation type and the city details of parents, make sure that the required reference data has been entered to the system through the Reference module.
  • It is mandatory to enter at least one contact detail to save the parent details. 
  • Donation panel is common to all the users (father, mother, guardian).
  • The Attendance progress bar is used to display a summary of the student's attendance details for a selected year.
  • The Religious Activities progress bar displays a summary of the student's religious activities for the selected year.
  • The Academic Life progress bar is used to display a summary of the student's academic life for the selected year.
  • Clicking the Home link will take you to the Homepage.
  • Clicking the Student Search link will take you to the Student Search page.

Add Parent or Guardian Details

  1. Select the relationship (father, mother or guardian). Note that this is mandatory.
  2. Enter the parent/guardian details. 
  3. Click Save to save the details or click Cancel to cancel the add operation.

Edit Parent/Guardian Details

  1. In the Parent Details page, make changes as required.
  2. Click Save to save the changes or click Cancel to cancel the changes.

Donation Details

Add Donations

  1. To enter donation details, click the Add New Donation button. The Add New Donation section will be displayed.
  2. Enter the donation details. 
  3. Click Save to save the donation details or click Cancel to cancel the changes.

Edit Donations

  1. Click the Edit button for the relevant donation.
  2. Make changes as required.
  3. Click Save to save the changes or click Cancel to cancel the changes.

Delete Donations

  1. To delete donation details, click the Delete button for the relevant donation.
  2. A confirmation message is displayed. Click OK to delete the donation details or click Cancel to cancel the deletion.

Subscribe to SMS notifications

  1. Select the SMS Service checkbox.
  2. Select the Package Type from the drop-down list. 
  3. Click Save to confirm the subscription.
Note:
  • Parents must enter their mobile number before subscibing to the SMS notifications.
  • It is mandatory to select the package type. 
  • Parents will be notified via SMS when they are subscribed or unsubscribed. 
  • Parents of past students will be automatically unsubscribed from SMS notifications.
