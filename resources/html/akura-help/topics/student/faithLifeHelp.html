Religious Activities

Navigation: Select Student Module->Search Student->Edit->Religious Activities

Using this page, you can manage student religious activity information.

Note: 

  • Fields marked with an asterisk (*) are mandatory.
  • If there are any existing religious activities, the data will be displayed in the page.
  • Before adding value information make sure that the required reference data has been entered to the system through the Reference module.
  • The Attendance progress bar is used to display a summary of the student's attendance details for the selected year.
  • The Religious Activities progress bar is used to display a summary of the student's religious activities for the selected year.
  • The Academic Life progress bar is used to display a summary of the student's academic life for the selected year.
  • Clicking the Home button will take you to the Homepage.
  • Clicking the Search Student button will take you to the Student Search page.

Add Religious Activities

  1. To add religious activities, click the Add button and fill in the required details.
  2. Click Save to save the details or click Cancel to cancel the add operation.

Edit Religious Activity Details

  1. To edit religious activity details, click the Edit button in the row section.
  2. Make changes as required.
  3. Click Save to save the changes or click Cancel to cancel the changes.

Delete Religious Activities

  1. To delete religious activity details, click the Delete button in the row section.
  2. A confirmation message is displayed. Click OK to delete the religious activity details or click Cancel to cancel the deletion.
