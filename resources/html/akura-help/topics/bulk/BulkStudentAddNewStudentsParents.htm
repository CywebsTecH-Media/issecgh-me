Student Uploads

Navigation: Select Bulk Upload Module-> Student Uploads

Using this page, you can upload student and parent related information to the system.

Add New Students

       File Upload Guidelines
  • The user must always use the Add New Students Template.xls template which can be downloaded from the Templates page.
    [Home ->Bulk Upload Module -> Templates]
  • Following fields marked in Red are mandatory: 
    Admission No (should have a unique value), Full Name, Name with Initials, Last Name, Gender, 1st day at school, Study Medium
  • Data should be available in the system when filling the following fields:
    Religion, Race, Study Medium, Nationality, Blood Group, School House, Method of Travel (1), Method of Travel (2), Method of Travel (3), Permanent City, Temporary City, Studied Medium
  • The country should be entered according to the Country drop-down list which can be found in the Add New Student page.
  • The date format should be mm/dd/yyyy.
  • All the phone numbers should be entered according to the following format:
    <country code without '+'> <phone number without '0'>
    E.g.: 0777597233 --> 94777597233
  • The capacity of the XLS file should not exceed 2MB.

  1. Browse the XLS file.
  2. Click the Upload button to upload data to the system.


Add New Parents

       File Upload Guidelines
  • The user must always use the Add New Parents Template.xls template which can be downloaded from the Templates page.
    [Home ->Bulk Upload Module -> Templates]
  • Following fields marked in Red are mandatory: 
    Student Admission No, Relationship, Full Name, NIC/SSN Number,
    Phone (Res)/Phone (Mob)/Email (personal) - either one or more can be filled out of these three fields.
  • Student Admission No must be a valid and an existing number.
  • Data should be available in the system when filling the following fields:
    Religion, Employment Status, Working Segment, City
  • All the phone numbers should be entered according to the following format:
    <country code without '+'> <phone number without '0'>
    E.g.: 0777597233 --> 94777597233
  • The capacity of the XLS file should not exceed 2MB.

  1. Browse the XLS file.
    • Click the Upload button to upload data to the system.


    Update Student Details

    1. Click the Export Student Details button.
    2. Select the following from the drop-down lists:
      • Select Year
      • Grade
      • Class
      • Study Medium
    3. Select the fields from the Available Fields list box.
    4. Click the right arrow button to add fields to the Selected Fields list box. (Note that you can remove fields from the Selected Fields list by clicking the left arrow button.)
    5. Click the Export button to download the existing student details spreadsheet with the required fields or click the Cancel button to cancel the operation.
      6.        File Upload Guidelines
      • The user must always use the spreadsheet downloaded according to the steps given above.
      • Following fields marked in Red are mandatory: 
        Admission No, Full Name, Name with Initials, Last Name, Gender, 1st Day at School, Study Medium
      • Student Admission No must be a valid and an existing number.
      • All the phone numbers should be entered according to the following format:
        <country code without '+'> <phone number without '0'>
        E.g.: 0777597233 --> 94777597233
      • The capacity of the XLS file should not exceed 2MB.

    6. Browse the updated XLS file.
    7. Click the Upload button to upload data to the system.

    Note:
    • Fields marked with an asterisk (*) are mandatory.
    • Actions can be performed by the admin or customized roles.
