User Login Uploads

Navigation: Select Bulk Upload Module-> User Login Uploads

Using this page, you can upload parent related login details to the system.

Add New User Login

       File Upload Guidelines
  • The user must always use the Add New Parent Login Template.xls template which can be downloaded from the Templates page.
    [Home ->Bulk Upload Module -> Templates]
  • Following fields marked in Red are mandatory: 
    userName, password, NIC/SSN Number, First Name, Last Name, Email Address
  • Unique values should be entered for the following fields:
    userName, NIC/SSN Number
  • All NIC/SSN Numbers should belong to existing parents in the databse.

  1. Browse the XLS file.
  2. Click the Upload button to upload data to the system.
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Actions can be performed by the admin or customized roles.
