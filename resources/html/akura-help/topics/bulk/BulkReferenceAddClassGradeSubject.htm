Reference Upload

Navigation: Select Bulk Upload Module-> Reference Uploads

Using this page, you can upload reference information to the system.

Add Classes

       File Upload Guidelines
  • The user must always use the Add New Classes Template.xls template which can be downloaded from the Templates page.
    [Home ->Bulk Upload Module -> Templates]
  • Unique values should be entered for the following mandatory field marked in Red:
    • Class
  • The capacity of the XLS file should not exceed 2MB.

  1. Browse the XLS.
  2. Click the Upload button to upload data to the system.

Add Grades

       File Upload Guidelines
  • The user must always use the Add New Grades Template.xls template which can be downloaded from the Templates page.
    [Home ->Bulk Upload Module -> Templates]
  • Following fields marked in Red are mandatory: 
    • Grade Name (should be a unique value)
    • Classes
  • If there are more than one Class in a Grade, the Grade should be entered repeatedly with each Class.
    GradeClass   
    Grade 1A
    Grade 1B  
    Grade 1C  

  • The capacity of the XLS file should not exceed 2MB.

  1. Browse the XLS file.
    • Click the Upload button to upload data to the system.

    Add Subjects

           File Upload Guidelines
    • The user must always use the Add New Subjects Template.xls template which can be downloaded from the Templates page.
      [Home ->Bulk Upload Module -> Templates]
    • Unique values should be entered for the following mandatory fields marked in Red:
      • Subject Name
      • Short Name
    • The capacity of the XLS file should not exceed 2MB.

    1. Browse the XLS file.
    2. Click the Uploadbutton to upload data to the system.

    Note:
    • Fields marked with an asterisk (*) are mandatory.
    • Actions can be performed by the admin or customized roles.
