Templates

Navigation: Select Bulk Upload Module-> Templates

Using this page, you can download the excel templates under each category.

Staff Related Templates

  1. Select the following check box:
    • Add New Staffs Template.xls
  2. Click the Download button to download the template from the system.

Student Related Templates

  1. Select the following check boxes:
    • Add New Students Template.xls
    • Add New Parents Template.xls
  2. Click the Download button to download templates from the system.

Reference Related Templates

  1. Select the following check boxes:
    • Add New Classes Template.xls
    • Add New Grades Template.xls
    • Add New Subjects Template.xls
  2. Click the Download button to download templates from the system.

User Login Related Templates

  1. Select the following check box:
    • Add New Parent Login Template.xls
  2. Click the Download button to download the template from the system.

Note:
  • Templates can also be downloaded by selecting the template.
  • Actions can be performed by the admin or customized roles.
