Generate Average Term Test Marks for Given Subjects for a Grade (class wise) Report

Navigation: Select Reporting Module->Student General Reports->Average Term Test Marks for  Given Subjects for a Grade

Using this page, you can generate a report of average term test marks for a selected subject for a selected grade.

  1. Select the following details:
    • Grade
    • Term
    • Subjects
  2. Click the Generate Report button.
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Mark entry for all subjects should be confirmed from the student module->Student Mark Entry to generate the report.
