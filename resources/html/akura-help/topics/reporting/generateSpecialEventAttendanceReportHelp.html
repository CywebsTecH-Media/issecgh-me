Student Attendance On Special Events

Navigation: Select Reporting Module->Student Attendance Reports->Per Day Class Wise Attendance Report.

Using this page, you can view student attendance for a selected event.

  1. Select the following details:
    • Event Name
    • Filtering option
  2. Click the Generate Report button.
Note: Fields marked with an asterisk (*) are mandatory.
