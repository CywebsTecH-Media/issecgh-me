User Wise SMS Subscription Report

Navigation: Select Reporting Module->SMS Reports->User Wise SMS Subscription Report

Using this page, you can generate user wise sms subscription report for a particular year - month. 

User Wise SMS Subscription Report

  1. Select the following details:
    • From, year
    • From, month
    • To, year
    • To, month
  2. Click the Generate Report button.
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • To be listed in the user wise sms subscription report, parents must fulfill the following criteria. 
    • sms subscription for any of the sms packages - low, medium, high.
