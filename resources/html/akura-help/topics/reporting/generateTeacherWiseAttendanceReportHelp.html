Generate Staff Wise Attendance Report

Navigation: Select Reporting Module->Staff Attendance Reports->Staff Wise Attendance Report.

Using this page, you can generate the staff wise attendance report.

  1. Select the following details:
    • Category
    • Registration No
    • Date range
  2. Click the Generate Report button.
Note: Fields marked with an asterisk (*) are mandatory.
