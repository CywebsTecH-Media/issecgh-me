Class Wise Student Marks Sheet

Navigation: Select Reporting Module->Student General Reports->Class Wise Student Marks Sheet

Using this page, you can generate the class wise student marks sheet.

  1. Select the following details:
    • Grade
    • Class
    • Term
    • Year
  2. Select an option to specify whether to sort by name or rank.
  3. Click the Generate Report button.
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Mark entry for all subjects should be confirmed from the student module->Student Mark Entry to generate the report.
