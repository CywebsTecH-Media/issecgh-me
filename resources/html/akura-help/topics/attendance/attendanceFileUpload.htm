Attendance File Upload

Navigation: Select Attendance Module-> Attendance File Upload

Using this page, you can upload staff attendance data to the system.

  1. Browse a CSV file.
  2. Click the Upload button to upload data to the system.
Note:
  • Fields marked with an asterisk (*) are mandatory.
  • Actions can be performed by the admin or customized roles.
  • The user should browse a CSV file according to the following column order.
    Date, ID, User Name, Department, Daily Schedule, First-In Time, Last-Out Time, Result, Early Duty (Sample), Work Time
  • The date format in the CSV file should be mm/dd/yyyy.
