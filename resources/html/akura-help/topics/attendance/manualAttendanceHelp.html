Student Attendance

Navigation: Select Attendance Module->Student Attendance

Using this page, you can manually enter class-wise student attendance.

Student Attendance

  1. Select the following details:
    • Class
    • Date
  2. Click the Search Class button to view the list of students for the selected class.
  3. Mark attendance using the check boxes. If necessary, click the Check All button to select all the check boxes.
  4. Click the Confirm Attendance check box, if it is required to send SMS notifications to the parents. 
  5. Click the Save button.
  6. A confirmations message is displayed. Click OK to save the changes and send SMS notifications or click Cancel to save attendance without sending the SMS notifications.
Notes:
  • Past students, suspended students and students on leave are not listed to be marked for attendance.
  • Parents must scubscribe for an SMS package to recieve SMS notifications. 
  • Only the parents of the absentees are notified via SMS.
  • SMS notifications are sent only from the allocated individual (parent) SMS quota.
  • Fields marked with an asterisk (*) are mandatory.
  • Attendance can only be marked for the current date and four dates before the current date. 
  • Dates does not include weekends or holidays. 
  • Holidays should be defined through the reference module, before marking attendance.
